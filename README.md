# Calculadora Web

Uma calculadora web desenvolvida com **HTML, CSS e JavaScript puro**, com foco em praticar manipulação do DOM, eventos e lógica de programação.

---

## 🚀 Funcionalidades

* [x] Operações básicas: soma, subtração, multiplicação e divisão
* [x] Interface interativa com botões
* [x] Exibição do cálculo atual
* [x] Atualização em tempo real no display
* [ ] Suporte ao teclado *(em desenvolvimento)*
* [ ] Histórico de cálculos *(em desenvolvimento)*
* [ ] Botão de apagar (←) *(em desenvolvimento)*

---

## 🛠️ Tecnologias utilizadas

* HTML5
* CSS3
* JavaScript
